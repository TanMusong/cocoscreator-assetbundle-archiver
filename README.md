# CocosCreator AssetBundle Packager

### Requirements

* CocosCreator `2.4.x (Test in 2.4.8)`

### Install
```bash
npm install -g asset-bundle-packager
```
### Usage

```bash
abpkg -- \
    --project <project path> \
    --platform [ios|android] \
    --key <assetbundle key> \
    --xxtea <xxtea key> \
    --mode [debug|release] 
```
